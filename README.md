# Movie Recommendation System

A Flask web app that recommends movies based on KNN similarity and predicts ratings using a Random Forest model. Posters are fetched dynamically from TMDB API.

## Features
- Movie search with interactive search bar
- Recommendations displayed with movie poster and predicted rating
- Movie-themed background and card layout

## How to Run
1. Clone the repo: `git clone https://github.com/GalibAmin/Movie-Rating-And-Recomendation.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `python app.py`
4. Open: `http://127.0.0.1:5000/`

Note-KNN(Movies_Rating_and_Recommendation).ipynb is code that will run on google collab for knn and randomforest model and all the pickles files so dont include this file on your working environment delete this file if u are working on vs-code (optional u can download and run on google collab to see the working mechanism of the project models).Rest everything will be same.

## Dependencies
- Flask
- pandas
- scikit-learn
- requests
